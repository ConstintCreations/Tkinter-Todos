# Tkinter Todos

This is a simple Todo app made with Tkinter in Python. You can create, complete, delete, and clear tasks. Deleting and clearing tasks will both ask for confirmation.

![Tkinter Todos Screenshot](./attachments/TkinterTodosApp.png)<br>
(Note that styling has only been tested on Windows)

## Usage

The package is on [TestPyPi](https://test.pypi.org/project/RP6-Tkinter-Todos/)

To install, you can use
```
pip install -i https://test.pypi.org/simple/ RP6-Tkinter-Todos
```

## Keyboard Shortcuts

Esc - Exit <br>
Enter - Add a task <br>
Tab - Cycle Focus <br>
A - Focus the add task entry <br>
T - Toggle the completion of the selected task <br>
X - Delete the selected task <br>
C - Clear Tasks
