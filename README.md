# Tkinter Todos

This is a simple Todo app made with Tkinter in Python. Note that styling has only been tested on Windows.

![Tkinter Todos Screenshot](./attachments/TkinterTodosApp.png)

## Usage

## Keyboard Shortcuts

Esc - Exit <br>
Enter - Add a task <br>
Tab - Cycle Focus <br>
A - Focus the add task entry <br>
T - Toggle the completion of the selected task <br>
X - Delete the selected task <br>
C - Clear Tasks
